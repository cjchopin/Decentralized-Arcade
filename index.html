Blockchain Arcade

How to play:

Install Metamask browser extension to play competetive mode.
Use practice mode to warm up.
If you completed a run in competetive mode, your rank (if high enaugh) will show up in the leaderboard.
One run costs 0.001 Ether (+gas fee's).
Careful: If you insert your money and reload the website your try is gone.
Once the timer runs out, Metamask will get your score and add it to the blockchain, you need to accept this request too.
